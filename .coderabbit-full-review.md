# CodeRabbit Full Repository Review

Complete setup for reviewing the entire ABCHR codebase via orphan-branch PR. Each phase stays under 300 files (Pro limit).

---

## Create the PR

1. Ensure `empty-branch` exists (see Setup below)
2. Go to: https://github.com/dev-noaman/ABCHR/compare/empty-branch...main
3. **base:** `empty-branch` | **compare:** `main`
4. Title: `Full Repository Review`
5. Description: `@coderabbitai full review`
6. Create pull request

## How to Run Each Phase

```powershell
# Copy phase config to .coderabbit.yaml
Copy-Item .coderabbit-phases/phaseN-*.yaml .coderabbit.yaml

# Commit and push
git add .coderabbit.yaml
git commit -m "CodeRabbit Phase N"
git push origin main

# On the PR, comment: @coderabbitai full review
```

---

## Setup: Create Orphan Branch (One-Time)

```powershell
git checkout main
git pull origin main
git checkout --orphan empty-branch
git rm -rf .
git commit --allow-empty -m "Empty branch for full CodeRabbit review"
git push -u origin empty-branch
git checkout main
```

---

## Cleanup (After All Phases)

- Close the PR (**do not merge**)
- Run:
  ```powershell
  git branch -D empty-branch
  git push origin --delete empty-branch
  ```
